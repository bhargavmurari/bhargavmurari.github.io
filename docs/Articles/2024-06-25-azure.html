Problem Statement#

Ensuring efficient and timely delivery of automation reports to stakeholders to facilitate informed decision-making and enable effective collaboration.

Solution Overview#

The solution is structured in multiple stages:

Stage 1: Create Azure App#

Details on how to set up and configure the Azure App for report automation.

Stage 2: Upload The Reports#

Instructions on uploading the automation test reports to the Azure App.

Stage 3: Prepare to Send Email#

Set up email automation with the following SMTP configuration:

smtp-util --sender:email@yourDomain.com --toList:emailid1;emailid2 --subject:Automation_TestReport --htmlFile:C:\\Automation\\API\\src\\main\\java\\Utilities\\AllureReportGenerator\\emailReport.html --displayName:K_Fleet_Desktop Automation Results --attachments:C:\\Automation\\API\\src\\main\\java\\Utilities\\AllureReportGenerator\\emailReport.html --port:YOUR_SMTP_PORT --host:YOUR_SMTP_HOST --userID:YOUR_SMTP_USERNAME --password:YOUR_SMTP_PASSWORD
